# Creating Courses

Admin Creators build course content from scratch. This guide walks through the full course creation flow.

---

## Course Creation Flow

The typical process is: create a course, add chapters, add pages to each chapter, embed tests if needed, then submit for review.

---

## Step 1 -- Create a Course

1. Go to **Courses** from the sidebar.
2. Click **Create Course**.
3. Fill in the course details.

| Field | Required | Description |
|---|---|---|
| **Title** | Yes | Course name shown to students |
| **Description** | Yes | Overview and learning objectives |
| **Category** | Yes | The course category |
| **Thumbnail** | Yes | Cover image for the course card |
| **Public** | No | Whether the course appears in the public catalog |

4. Click **Create**.

The course starts in **Draft** status.

---

## Step 2 -- Add Chapters

1. Open the course.
2. Click **Add Chapter**.
3. Enter a chapter title and optional description.
4. Drag chapters to reorder them.

---

## Step 3 -- Add Pages

1. Click into a chapter.
2. Click **Add Page**.
3. Use the content editor to add text, images, and media.
4. Reorder pages by dragging.

---

## Step 4 -- Embed Tests

You can embed a test inside any page to create an in-course assessment.

1. Open a page.
2. Click **Embed Test**.
3. Select an existing test or create a new one.

---

## Step 5 -- Submit for Review

When the course is ready:

1. Click **Submit for Review**.
2. The course moves to **Pending Review** status.
3. An Admin Publisher will review it.

If changes are requested, the course returns to you with feedback. Make the changes and resubmit.

{% hint style="info" %}
Courses stay in Draft until you submit them. You can save and come back at any time.
{% endhint %}
