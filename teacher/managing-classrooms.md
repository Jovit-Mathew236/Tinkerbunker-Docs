# Manage Classrooms

---

## Create a Classroom

1. Go to **Classroom** in the navigation bar.
2. Click **Create Classroom**.
3. Enter a name and configure the details.
4. Add students and link courses.

![Create Classroom](../.gitbook/assets/create-classroom.png)

---

## What's in a Classroom

| Element | Description |
|---|---|
| **Students** | The learners assigned to this classroom |
| **Courses** | Courses linked for the class to access |
| **Teacher** | You — the classroom creator |

---

## Edit a Classroom

Open a classroom and click **Edit** to update the name, students, or linked courses.

{% hint style="info" %}
Students see only the classroom they are assigned to.
{% endhint %}
