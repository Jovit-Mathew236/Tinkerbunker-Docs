# 🏫 Manage Classrooms

A classroom holds your students and courses together.

---

## ➕ Create a Classroom

1. Go to **Classroom** in the nav bar
2. Click **Create Classroom**
3. Enter a **Name** (e.g. "Grade 10 — Science")
4. Add an optional **Description**
5. Click **Create**

<figure><img src="../.gitbook/assets/create-classroom.png" alt="Create Classroom Form"><figcaption></figcaption></figure>

---

## 👥 Add Students

1. Open your classroom
2. Go to the **Students** tab
3. Click **Add Students** — search by name or email
4. Select and confirm

| Column | Shows |
| --------------- | -------------------------------- |
| **Name** | Student's display name |
| **Email** | Registered email |
| **Status** | Active or inactive |
| **Enrolled On** | Date added |

{% hint style="info" %}
Removing a student only removes them from the roster — not from courses they already accessed.
{% endhint %}

---

## 📚 Link Courses

1. Open your classroom → **Courses** tab
2. Click **Link Course**
3. Search, select, and link

{% hint style="info" %}
Linking a private course gives all students in the classroom access to it.
{% endhint %}

---

## 📋 Classroom Overview

| Tab | What's inside |
| ---------------- | ------------------------------------------ |
| **Summary** | Name, description, student & course counts |
| **Students** | Full roster with management options |
| **Courses** | Linked courses — link or unlink |
| **Assignments** | Tests assigned to this classroom |
| **Activity** | Recent activity log |
