# Managing Classrooms

Classrooms are the organizational backbone of TinkerBunker for teachers. You can create classrooms, add students, link courses, and manage all classroom settings from a single interface.

---

## Accessing Classroom Management

1. Click **Classroom** in the navigation bar.
2. You will see a list of all classrooms you have created.
3. Click on any classroom to manage it, or click **Create Classroom** to set up a new one.

![Classroom Management](.gitbook/assets/teacher-classroom-list.png)

---

## Creating a Classroom

{% tabs %}
{% tab title="Steps" %}
1. Click **Classroom** in the navigation bar.
2. Click the **Create Classroom** button.
3. Fill in the classroom details:
   - **Name** — A descriptive name for the classroom (e.g., "Grade 10 - Science").
   - **Description** — Optional description of the classroom's purpose.
4. Click **Create** to finalize.
5. The new classroom appears in your classroom list.
{% endtab %}

{% tab title="Screenshot" %}
![Create Classroom Form](.gitbook/assets/create-classroom.png)
{% endtab %}
{% endtabs %}

## Editing a Classroom

1. Open the classroom you want to edit.
2. Click the **Edit** button or gear icon.
3. Modify the classroom name, description, or other settings.
4. Click **Save** to apply changes.

{% hint style="info" %}
Editing a classroom name or description does not affect the students or courses already linked to it.
{% endhint %}

---

## Assigning Students

Students are added to a classroom so they can access linked courses and receive assignments.

{% tabs %}
{% tab title="Steps" %}
1. Open the classroom.
2. Navigate to the **Students** tab.
3. Click **Add Students**.
4. Search for students by name or email.
5. Select the students you want to add.
6. Click **Add** to confirm.
{% endtab %}

{% tab title="Bulk Add" %}
If your institution supports it, you may be able to add students in bulk by uploading a list or syncing with a student management system.
{% endtab %}
{% endtabs %}

### Student List

Once students are assigned, the classroom shows:

| Column | Description |
| -------------- | ------------------------------------------- |
| **Name** | Student's display name. |
| **Email** | Student's registered email. |
| **Status** | Active or inactive. |
| **Enrolled On**| Date the student was added to the classroom. |

{% hint style="warning" %}
Removing a student from a classroom does not unenroll them from courses they have already accessed. It only removes them from the classroom roster and future assignments.
{% endhint %}

## Linking Courses

Linking a course to a classroom makes it accessible to all students in that classroom, including private courses.

{% tabs %}
{% tab title="Steps" %}
1. Open the classroom.
2. Navigate to the **Courses** tab.
3. Click **Link Course**.
4. Browse or search for the course you want to link.
5. Select the course and click **Link**.
6. The course now appears in the classroom's course list and is visible to students.
{% endtab %}

{% tab title="Unlinking" %}
1. Open the classroom.
2. Navigate to the **Courses** tab.
3. Click the **Unlink** button next to the course you want to remove.
4. Confirm the action.

Students who have already started the course retain their progress, but the course will no longer appear in the classroom view.
{% endtab %}
{% endtabs %}

{% hint style="info" %}
Linking a private course to a classroom is the primary way to give students access to private content without sharing individual links.
{% endhint %}

## Classroom Overview

The classroom detail page provides a consolidated view:

| Section | Description |
| --------------- | -------------------------------------------------------------- |
| **Summary** | Classroom name, description, student count, course count. |
| **Students** | Full list of enrolled students with management options. |
| **Courses** | Linked courses with options to link or unlink. |
| **Assignments** | Tests assigned to this classroom with statuses. |
| **Activity** | Recent activity log for the classroom. |

---

## Best Practices

{% hint style="info" %}
- Use clear, descriptive names for classrooms so students can easily identify them.
- Link only relevant courses to avoid overwhelming students with unrelated content.
- Review the student list periodically to remove inactive students.
- Use classroom-level assignments to ensure all students in a class receive the same tests.
{% endhint %}
