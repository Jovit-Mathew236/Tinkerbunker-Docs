# Browsing Courses

As a teacher on TinkerBunker, you have access to the full course catalog. You can browse courses, view their content and structure, and favorite courses you want to reference or link to your classrooms.

---

## Accessing the Course Catalog

1. Click **Courses** in the navigation bar.
2. The course catalog loads with all available courses.
3. Use the search bar to find courses by title or keyword.
4. Apply filters to narrow results by category, subject, or other criteria.

![Course Catalog - Teacher View](.gitbook/assets/teacher-course-catalog.png)

---

## Viewing Course Details

Click on any course card to open its detail page. The detail page shows:

| Field | Description |
| ---------------------- | ------------------------------------------------------------ |
| **Title** | Name of the course. |
| **Description** | Full overview of the course content and objectives. |
| **Chapters** | List of chapters with page counts. |
| **Duration** | Estimated time to complete. |
| **Enrolled Students** | Number of students currently enrolled. |
| **Visibility** | Public or Private. |
| **Price** | Free or the amount in INR. |

{% hint style="info" %}
As a teacher, you can browse course content to evaluate whether it is suitable for your students before linking it to a classroom.
{% endhint %}

---

## Course Structure Preview

From the detail page, you can expand chapters to see the list of pages within each one. This gives you a full picture of the course content without enrolling.

```
Course: Introduction to Physics
├── Chapter 1: Mechanics
│   ├── Page 1: Newton's Laws
│   ├── Page 2: Motion and Forces
│   └── Page 3: Practice Problems
├── Chapter 2: Thermodynamics
│   ├── Page 1: Heat and Temperature
│   └── Page 2: Laws of Thermodynamics
└── Chapter 3: Waves
    ├── Page 1: Sound Waves
    └── Page 2: Light and Optics
```

## Favoriting Courses

You can favorite courses to save them for quick access later. See the [Favorites](favorites.md) guide for detailed instructions.

- Click the **heart icon** on any course card or detail page.
- Favorited courses are accessible from the **Favorites** tab on the Courses page.

---

## Linking a Course to a Classroom

When you find a course suitable for your students:

1. Note the course or open its detail page.
2. Navigate to **Classroom** → select your classroom → **Courses** tab → **Link Course**.
3. Search for and select the course.
4. Link it to make it accessible to all students in that classroom.

For full details, see [Managing Classrooms](managing-classrooms.md).

{% hint style="warning" %}
Linking a paid course to a classroom does not bypass the payment requirement for students. Students will still need to pay for paid courses individually unless an institutional arrangement exists.
{% endhint %}

## Filtering and Search

| Feature | Description |
| ------------- | ------------------------------------------------------------ |
| **Search** | Type keywords to search course titles and descriptions. |
| **Category** | Filter by subject area or category. |
| **Visibility**| Filter by Public or Private courses. |
| **Price** | Filter by Free or Paid courses. |

{% hint style="info" %}
Use multiple filters together to quickly find the most relevant courses for your teaching needs.
{% endhint %}
