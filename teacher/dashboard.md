## 🧭 Navigation

| Icon | Tab | What it does |
| ---- | ------------- | --------------------------------- |
| 🏫 | **Classroom** | Create & manage your classrooms |
| 📚 | **Courses** | Browse and favorite courses |
| 📝 | **Tests** | Create tests, assign & grade them |
| 📊 | **Stats** | View performance analytics |

## 👀 What You See

| Section | At a glance |
| -------------------- | ---------------------------------------- |
| **My Classrooms** | Student count, linked courses per class |
| **Recent Activity** | Tests created, assignments sent, graded |
| **Upcoming** | Deadlines, pending evaluations, quizzes |
| **Notifications** | Submissions, announcements, updates |

---

## ⚡ Quick Actions

1. **Create a Classroom** — set up a new class
2. **Create a Test** — build a test from scratch
3. **Assign a Test** — send a test to students
4. **Start a Remote Quiz** — launch a live session
