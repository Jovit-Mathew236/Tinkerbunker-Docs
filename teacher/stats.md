# Teacher Stats

The **Stats** section provides teachers with a comprehensive analytics dashboard to monitor student performance, track engagement, and measure the effectiveness of their teaching.

---

## Accessing Teacher Stats

1. Click **Stats** in the navigation bar.
2. Your teacher statistics dashboard loads.

![Teacher Stats Dashboard](../.gitbook/assets/teacher-stats.png)

---

## Dashboard Sections

| Section | Description |
| ------------------------------- | ------------------------------------------------------------------------------ |
| **Classroom Performance** | Aggregated scores and completion rates per classroom. |
| **Test Analytics** | Performance breakdowns for each test you have created and assigned. |
| **Student Engagement** | Metrics on student activity — logins, course progress, test attempts. |
| **Assignment Overview** | Summary of all assignments with submission and evaluation status. |
| **Comparative Metrics** | Cross-classroom and cross-test comparisons. |

---

## Classroom Performance

View how each of your classrooms is performing:

| Metric | Description |
| ----------------------------- | ----------------------------------------------------------- |
| **Average Score** | Mean test score across all students in the classroom. |
| **Completion Rate** | Percentage of students who have completed assigned courses. |
| **Active Students** | Number of students who have been active in the last 7 days. |
| **Top Performers** | Students with the highest aggregate scores. |

{% hint style="info" %}
Click on a classroom card to drill down into individual student performance within that classroom.
{% endhint %}

---

## Test Analytics

For each test you have assigned, detailed analytics are available:

| Metric | Description |
| ----------------------------- | ---------------------------------------------------------------- |
| **Average Score** | Mean score across all students who took the test. |
| **Highest / Lowest Score** | Score range for the test. |
| **Question-Level Analysis** | Percentage of students who answered each question correctly. |
| **Difficulty Index** | Identifies which questions were hardest and easiest. |
| **Discrimination Index** | Shows which questions effectively differentiate strong and weak students. |

{% tabs %}
{% tab title="Overall View" %}
The overall test analytics page shows summary cards for each test with key metrics.
{% endtab %}

{% tab title="Question Breakdown" %}
Click on a specific test to see the question-level breakdown:

| Question | Correct % | Avg. Time | Difficulty |
| ---------- | --------- | --------- | ---------- |
| Q1 | 85% | 30s | Easy |
| Q2 | 42% | 55s | Hard |
| Q3 | 71% | 40s | Medium |
| Q4 | 90% | 20s | Easy |
{% endtab %}
{% endtabs %}

---

## Student Engagement

Track how actively your students are using TinkerBunker:

| Metric | Description |
| ---------------------- | ------------------------------------------------------------- |
| **Active Students** | Students who logged in within the last 7 days. |
| **Course Progress** | Average progress across all students for linked courses. |
| **Test Participation** | Percentage of students who have attempted assigned tests. |
| **Time Spent** | Average time students spend on the platform per session. |

{% hint style="warning" %}
Low engagement metrics may indicate students need additional motivation or that the content needs to be more accessible. Consider reaching out to students who have not been active.
{% endhint %}

---

## Assignment Overview

A consolidated view of all your assignments:

| Column | Description |
| ---------------------- | ---------------------------------------------------- |
| **Assignment Title** | Name of the test/assignment. |
| **Classroom** | Which classroom it was assigned to. |
| **Due Date** | Deadline for submission. |
| **Submissions** | Number of students who have submitted. |
| **Evaluated** | Number of submissions you have graded. |
| **Avg. Score** | Mean score of evaluated submissions. |

---

## Comparative Metrics

Compare performance across different classrooms or tests:

{% tabs %}
{% tab title="Cross-Classroom" %}
- Compare average scores between classrooms.
- Identify classrooms that may need more attention.
- View side-by-side charts of engagement and performance.
{% endtab %}

{% tab title="Cross-Test" %}
- Compare student performance across different tests.
- Identify topics that students consistently struggle with.
- Use insights to adjust your teaching strategy.
{% endtab %}
{% endtabs %}

{% hint style="info" %}
Use comparative metrics to identify patterns. If one classroom consistently underperforms on a specific topic, consider providing additional resources or review sessions for that group.
{% endhint %}

---

## Exporting Data

You can export stats data for offline analysis:

| Export Format | Available For |
| ------------- | --------------------------------------------- |
| **CSV** | Test results, student scores, engagement data. |
| **PDF** | Summary reports for classroom performance. |

Click the **Export** button on any analytics page to download the data.
