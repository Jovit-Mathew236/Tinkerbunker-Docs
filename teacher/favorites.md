# Favorites

Teachers can **favorite** courses on TinkerBunker to bookmark them for quick access. This is useful for organizing courses you frequently reference, plan to link to classrooms, or want to recommend to students.

---

## How to Favorite a Course

{% tabs %}
{% tab title="From the Course Catalog" %}
1. Navigate to **Courses** in the navigation bar.
2. Browse or search for a course.
3. Click the **heart icon** on the course card.
4. The icon fills in, confirming the course is now favorited.
{% endtab %}

{% tab title="From the Course Detail Page" %}
1. Open a course by clicking on its card.
2. Click the **heart icon** near the course title on the detail page.
3. The course is added to your favorites.
{% endtab %}
{% endtabs %}

![Favorite Course - Teacher](.gitbook/assets/teacher-favorite-course.png)

---

## How to Unfavorite a Course

1. Locate the favorited course (catalog, detail page, or favorites list).
2. Click the **filled heart icon** to toggle it off.
3. The course is removed from your favorites.

{% hint style="info" %}
Unfavoriting a course does not affect any classroom links. If the course is linked to a classroom, it remains linked regardless of your favorite status.
{% endhint %}

---

## Where to Find Favorited Courses

| Location | How to Access |
| -------------------- | -------------------------------------------------------------- |
| **Courses Page** | Switch to the **Favorites** tab to see all your favorited courses. |
| **Course Catalog** | Favorited courses display a filled heart icon on their cards. |

---

## Use Cases for Teachers

| Use Case | Description |
| ------------------------------- | ------------------------------------------------------------------- |
| **Quick Reference** | Favorite courses you frequently demo or refer to during class. |
| **Classroom Planning** | Bookmark courses you plan to link to classrooms in the future. |
| **Course Recommendations** | Maintain a list of courses you recommend to students. |
| **Subject Organization** | Use favorites to group courses by subject for easy access. |

{% hint style="info" %}
There is no limit on the number of courses you can favorite. Use this feature freely to keep your teaching resources organized.
{% endhint %}
