# 📝 Create Tests

Build a test, add questions, and it's ready to assign or use in a live quiz.

<figure><img src="../.gitbook/assets/create-test.png" alt="Create Test"><figcaption></figcaption></figure>

---

## 🪜 Steps

1. Go to **Tests** → click **Create Test**
2. Fill in the basics:

| Setting | What it means |
| ---------------------- | -------------------------------------- |
| **Title** | Name your test (required) |
| **Description** | Short overview (optional) |
| **Duration** | Time limit in minutes (optional) |
| **Shuffle Questions** | Randomize order per student |
| **Show Results** | Let students see results after submit |

3. **Add questions** (see types below)
4. **Preview** — check the student view
5. **Save**

---

## ❓ Question Types

| Type | How it works |
| -------------------- | ---------------------------------------- |
| **MCQ (Single)** | One correct answer from multiple options |
| **MCQ (Multiple)** | Multiple correct answers allowed |
| **True / False** | Binary choice |
| **Fill in the Blank** | Student types a short answer |
| **Subjective** | Open-ended, you grade manually |

{% hint style="info" %}
Each question has its own marks. Subjective questions need manual evaluation.
{% endhint %}

---

## ✏️ Managing Questions

| Action | How |
| ------------- | ------------------------------------------ |
| **Reorder** | Drag and drop |
| **Edit** | Click the question to modify |
| **Delete** | Remove from the test |
| **Duplicate** | Copy a question for quick variations |
