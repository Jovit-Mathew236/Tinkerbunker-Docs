# Create Tests

---

## How to Create a Test

1. Go to **Tests** in the navigation bar.
2. Click **Create Test**.
3. Enter the test name and configure settings.
4. Add questions one by one.
5. Save the test.

![Create Test](../.gitbook/assets/create-test.png)

---

## Question Types

| Type | Description |
|---|---|
| **Multiple Choice** | Pick one or more correct answers |
| **True / False** | Two-option questions |
| **Fill in the Blank** | Students type the answer |
| **Subjective** | Open-ended, manually graded |

---

## Test Settings

| Setting | What It Does |
|---|---|
| **Time Limit** | Set a countdown timer for the test |
| **Shuffle Questions** | Randomize question order per student |
| **Show Results** | Let students see answers after submission |

{% hint style="info" %}
You can edit a test at any time before students start taking it.
{% endhint %}
