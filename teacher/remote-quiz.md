# 🎮 Live Remote Quiz

Run a real-time quiz in class. Students answer with physical remotes — scores update live.

<figure><img src="../.gitbook/assets/remote-quiz-session.png" alt="Remote Quiz Session"><figcaption></figcaption></figure>

---

## 🔌 What You Need

| Component | Purpose |
| -------------------- | ---------------------------------------- |
| **Student Remotes** | Handheld devices with numbered buttons |
| **USB Receiver** | Plugs into your computer, captures input |
| **Projector/Display** | Shows questions to the class |

{% hint style="warning" %}
Check all remote batteries before starting.
{% endhint %}

---

## 🛠️ Setup

1. Plug in the **USB receiver**
2. Go to **Tests → Remote Quiz**
3. Select (or create) a test
4. Configure:

| Setting | What it does |
| ---------------------- | ---------------------------------------- |
| **Time per question** | Seconds allowed per question |
| **Show correct answer** | Reveal answer after each question |
| **Scoring mode** | Standard or negative marking |

5. **Map remotes** — assign each remote ID to a student
6. Click **Start Session**

<figure><img src="../.gitbook/assets/remote-quiz-setup.png" alt="Remote Quiz Setup"><figcaption></figcaption></figure>

---

## ▶️ Running the Quiz

1. Question appears on screen with a countdown
2. Students press their answer button
3. Response counter shows progress in real time
4. Correct answer is revealed (if enabled)
5. Leaderboard updates — next question loads

<figure><img src="../.gitbook/assets/remote-quiz-live.png" alt="Real-Time Response Dashboard"><figcaption></figcaption></figure>

{% hint style="info" %}
You control the pace. Pause between questions to discuss answers.
{% endhint %}

---

## 🏁 After the Quiz

- Final leaderboard is shown
- Results auto-save to TinkerBunker
- Export results or review individual responses
