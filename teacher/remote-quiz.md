# Live Remote Quiz

---

## What Is It

A live, interactive quiz session run in the classroom using physical remote devices. Students answer questions in real time using handheld remotes.

---

## What You Need

| Item | Description |
|---|---|
| **Receiver** | A USB device connected to the teacher's computer |
| **Remotes** | Handheld clickers distributed to each student |

---

## How to Run a Session

1. Connect the receiver device to your computer.
2. Go to **Tests** and select a test.
3. Click **Start Remote Quiz**.
4. Students answer each question using their remotes.
5. View live responses on your screen.
