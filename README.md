# Welcome to TinkerBunker

TinkerBunker is a comprehensive EdTech platform designed for managing courses, classrooms, tests, and educational workflows across institutions. Whether you are a student looking to learn, a teacher managing classrooms, or an institute administrator overseeing operations, TinkerBunker provides the tools you need.

---

## What Can You Do with TinkerBunker?

- **Students** enroll in courses, take tests, earn certificates, and track progress.
- **Teachers** manage classrooms, create tests, assign coursework, and run remote quizzes.
- **Institutes** administer classrooms, approve users, manage teams, and monitor performance.
- **Guardians** link to students and monitor their learning progress.
- **Partners** manage schools, sub-partners, licensing, and billing.
- **Sales** teams manage partner relationships, coupons, and pricing.
- **Admin Creators** build and manage course content.
- **Admin Publishers** review, approve, and publish courses.
- **Super Admins** oversee the entire platform, including courses and categories.
- **Public Users** browse available courses and verify certificates without logging in.
