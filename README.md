# TinkerBunker Admin Documentation

Internal documentation for the TinkerBunker administrative team. This covers the Sales, Admin Creator, Admin Publisher, and Super Admin roles.

For user-facing documentation (Students, Teachers, Institutes, Partners, Guardians), see the main documentation site.

---

## Admin Roles

| Role | Responsibility |
|---|---|
| **Sales** | Manage partner relationships, coupons, and pricing |
| **Admin Creator** | Build and manage course content |
| **Admin Publisher** | Review, approve, and publish courses. Manage sales teams |
| **Super Admin** | Full platform oversight including courses, categories, and system settings |
