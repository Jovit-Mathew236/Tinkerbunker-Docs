# Screenshot Assets

Place your screenshots in this folder. The documentation references these filenames.

## How to Take Screenshots
1. Navigate to the relevant page in the app
2. Use browser DevTools to set viewport to **1280x800** for consistency
3. Save as PNG with the exact filename listed below
4. Place in this folder and commit
