# Test System

TinkerBunker includes a built-in test engine for creating assessments and tracking results.

---

## Test Types

| Type | Description |
|---|---|
| **Standalone Test** | A test assigned directly to a classroom |
| **Embedded Test** | A test embedded inside a course page |
| **Practice Test** | Optional, non-graded test for self-study |

---

## Question Types

| Type | How It Works |
|---|---|
| **Multiple Choice** | One correct answer from several options |
| **Multiple Select** | More than one correct answer |
| **True / False** | Two-option question |

---

## Grading

Tests are auto-graded immediately after submission. Results include:

- Total score and percentage.
- Correct and incorrect answers.
- Time taken.

Teachers can also manually grade assignments that include subjective questions.

---

## Who Can Create Tests

| Role | Can Create Tests |
|---|---|
| Teacher | Yes -- for their classrooms |
| Admin Creator | Yes -- embedded in courses |
