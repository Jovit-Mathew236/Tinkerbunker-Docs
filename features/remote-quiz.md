# Live Remote Quiz

---

Remote quiz is a live, interactive session where students answer questions in real time using physical handheld remotes.

---

## How It Works

| Step | What Happens |
|---|---|
| **Setup** | Teacher connects a USB receiver to their computer |
| **Distribution** | Students receive handheld remote clickers |
| **Start** | Teacher launches the quiz from the platform |
| **Response** | Students press buttons on their remotes to answer |
| **Results** | Teacher sees live responses on screen |

---

## Requirements

- A USB receiver device connected to the teacher's computer.
- Handheld remotes for each student.
- A test already created on the platform.

{% hint style="info" %}
Remote quiz sessions are managed by teachers and require physical hardware.
{% endhint %}
