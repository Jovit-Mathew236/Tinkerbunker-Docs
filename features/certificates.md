# Certificates

TinkerBunker automatically generates certificates when a student completes a course.

---

## How It Works

1. Student completes all chapters and required tests in a course.
2. A certificate is generated automatically.
3. The student can download it from their dashboard.
4. Anyone can verify the certificate using its unique ID.

---

## Certificate Details

Each certificate includes:

| Field | Description |
|---|---|
| **Student Name** | Name of the student |
| **Course Name** | The completed course |
| **Completion Date** | Date the course was finished |
| **Certificate ID** | Unique verification code |

---

## Public Verification

Certificates can be verified at the [public verification page](../public/verify-certificate.md) without logging in.
