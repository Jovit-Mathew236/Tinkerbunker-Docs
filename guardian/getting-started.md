# Getting Started as a Guardian

Guardians are parents or caretakers who link to students on the platform. As a guardian, you can monitor your child's academic progress, view test results, and stay informed about their learning journey.

---

## What Guardians Can Do

| Capability                  | Description                                                   |
| --------------------------- | ------------------------------------------------------------- |
| **Link to Students**        | Send a linkage request to connect with your child's account.  |
| **Monitor Progress**        | View course completion, module progress, and activity logs.   |
| **View Test Results**       | Access detailed results for quizzes, tests, and assessments.  |
| **Approve Linkage Requests**| Accept or decline linkage requests initiated by students.     |

{% hint style="info" %}
Guardians have read-only access. You cannot modify student data, course content, or classroom assignments.
{% endhint %}

---

## Creating Your Account

1. Download the TinkerBunker app or visit the web portal.
2. Click **Sign Up** and select **Guardian** as your role.
3. Fill in your details:

| Field         | Required | Description                        |
| ------------- | -------- | ---------------------------------- |
| **Full Name** | Yes      | Your name as it will appear to students and institutes. |
| **Email**     | Yes      | Used for login and notifications.  |
| **Phone**     | Yes      | Mobile number for verification.    |
| **Password**  | Yes      | Secure password for your account.  |

4. Verify your email or phone number.
5. Log in to access your Guardian Dashboard.

## Guardian Dashboard Overview

After logging in, you land on the Guardian Dashboard which shows:

- **Linked Students** — list of students you are connected to.
- **Pending Linkage Requests** — requests awaiting approval.
- **Recent Activity** — summary of recent student activity across all linked students.

{% hint style="warning" %}
If you have no linked students, the dashboard will prompt you to initiate a linkage request. See [Linking Students](linking-students.md) for detailed instructions.
{% endhint %}
