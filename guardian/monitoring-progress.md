# Monitoring Progress

Once you are linked to a student, you gain read-only access to their academic data. This section covers all the ways you can track your child's progress on the platform.

---

## Student Overview

1. Go to your Guardian Dashboard.
2. Click on a linked student's name to open their profile.

The student overview displays:

| Section              | Description                                                  |
| -------------------- | ------------------------------------------------------------ |
| **Active Courses**   | Courses the student is currently enrolled in.                |
| **Overall Progress** | Aggregated completion percentage across all courses.         |
| **Recent Activity**  | Last 10 actions (lessons opened, quizzes taken, etc.).       |
| **Upcoming Tests**   | Scheduled assessments with dates and subjects.               |

## Course Progress

To view detailed progress for a specific course:

1. Open the student's profile.
2. Click on a course under **Active Courses**.
3. The course progress view shows:

| Metric                  | Description                                         |
| ----------------------- | --------------------------------------------------- |
| **Modules Completed**   | Number of modules finished out of total.            |
| **Current Module**      | The module the student is currently working on.     |
| **Time Spent**          | Total time the student has spent on this course.    |
| **Completion %**        | Overall percentage of course content completed.     |
| **Last Accessed**       | When the student last opened this course.           |

{% hint style="info" %}
Progress data is updated in near real-time. There may be a short delay (up to a few minutes) between a student's action and the update appearing in your view.
{% endhint %}

### Module-Level Breakdown

Expand any module to see topic-level completion:

- Green checkmark — topic completed.
- Yellow circle — topic in progress.
- Gray circle — topic not started.

## Test Results

To view test and quiz results:

1. Open the student's profile.
2. Navigate to the **Tests** tab.
3. The test list shows:

| Column          | Description                                           |
| --------------- | ----------------------------------------------------- |
| **Test Name**   | Title of the assessment.                              |
| **Course**      | The course the test belongs to.                       |
| **Date Taken**  | When the student completed the test.                  |
| **Score**       | Points scored out of total.                           |
| **Percentage**  | Score as a percentage.                                |
| **Status**      | Completed, Missed, or Upcoming.                       |

### Detailed Test View

Click on any completed test to see:

- **Question-by-question breakdown** — each question, the student's answer, the correct answer, and whether it was marked correct.
- **Time per question** — how long the student spent on each question.
- **Comparison** — how the student's score compares to the classroom average (if available).

{% hint style="warning" %}
Some tests may have restricted result visibility based on institute or teacher settings. If results are not available, contact the institute for more information.
{% endhint %}

## Activity Timeline

The **Activity Timeline** provides a chronological view of the student's actions:

- Lessons opened
- Quizzes attempted
- Assignments submitted
- Courses started or completed
- Login timestamps

Use the date filter to narrow down the timeline to a specific period.

## Notifications

Guardians can opt in to receive notifications for key events:

| Notification Type         | Description                                        |
| ------------------------- | -------------------------------------------------- |
| **Test Completed**        | Sent when the student finishes a quiz or test.     |
| **Low Activity Alert**    | Sent if the student has not logged in for N days.  |
| **Course Completed**      | Sent when the student finishes all modules.        |
| **Score Below Threshold** | Sent if a test score falls below a set percentage. |

To configure notifications:

1. Go to **Guardian Settings → Notifications**.
2. Toggle the notification types you want to receive.
3. Set thresholds where applicable (e.g., inactivity days, score threshold).

{% hint style="info" %}
Notification preferences are per-student. You can configure different settings for each linked student.
{% endhint %}
