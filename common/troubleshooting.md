# 🛠️ Troubleshooting

Quick fixes for common issues.

---

| Problem | Solution |
|---------|----------|
| Can't log in | Check email/password. Use **Forgot Password** to reset. |
| Email verification not received | Check spam folder. Resend from the login page. |
| Payment failed | Retry. Check bank balance or try a different method. |
| Course not showing after payment | Refresh the page. If still missing, contact support. |
| Certificate not generated | Ensure all modules are marked complete. |
| Can't access a course | Check if your enrollment is active and approved. |
| Account stuck on "Pending Approval" | Contact your institute or partner admin. |
| Video not playing | Check internet connection. Try a different browser. |
| Quiz won't submit | Check your connection and retry. Don't close the tab. |
| Page not loading | Clear browser cache or try incognito mode. |
