# Glossary

| Term | Definition |
|---|---|
| **Admin Creator** | Role responsible for building course content |
| **Admin Publisher** | Role responsible for reviewing and publishing courses |
| **Approved** | Course status meaning it has passed review and is ready to publish |
| **Category** | A grouping used to organize courses in the catalog |
| **Changes Requested** | Course status meaning the publisher sent it back for edits |
| **Coupon** | A discount code that can be applied to seat purchases |
| **Draft** | Course status meaning it is still being built |
| **Institute** | A school or organization on the platform |
| **Partner** | An organization that manages one or more institutes |
| **Pending Review** | Course status meaning it has been submitted for publisher review |
| **Published** | Course status meaning it is live and available for enrollment |
| **Razorpay** | The payment gateway used for processing transactions |
| **Sales** | Role responsible for managing partner relationships and pricing |
| **Seat** | A license for one student to access one course |
| **Super Admin** | Role with full platform access and override capabilities |
| **White Labeling** | Customizing the platform's branding for a partner or institute |
