# Taking Courses

Once you are enrolled in a course, you can start learning right away. Courses on TinkerBunker follow a structured hierarchy to keep content organized and easy to navigate.

---

## Course Structure

Every course is organized into **Chapters**, and each chapter contains one or more **Pages**.

```
Course
├── Chapter 1
│   ├── Page 1
│   ├── Page 2
│   └── Page 3
├── Chapter 2
│   ├── Page 1
│   └── Page 2
└── Chapter 3
    ├── Page 1
    ├── Page 2
    └── Page 3
```

| Level | Description |
| ----------- | -------------------------------------------------------------------------- |
| **Course** | The top-level container. Contains a title, description, and chapters. |
| **Chapter** | A logical grouping of related content. Think of it as a module or unit. |
| **Page** | The individual content unit. Can include text, images, videos, and media. |

---

## Navigating Course Content

{% tabs %}
{% tab title="Starting a Course" %}
1. Go to **Dashboard → My Courses** or the **Courses** page.
2. Click on the course you want to continue.
3. You will land on the course overview showing all chapters.
4. Click on a chapter to expand it, then click a page to begin.
{% endtab %}

{% tab title="Navigating Between Pages" %}
- Use the **Next** and **Previous** buttons at the bottom of each page to move sequentially.
- Use the sidebar chapter/page list to jump to any specific page directly.
- Your current position is highlighted in the sidebar.
{% endtab %}

{% tab title="Resuming a Course" %}
TinkerBunker remembers where you left off. When you re-open a course, you are taken directly to the last page you were viewing.
{% endtab %}
{% endtabs %}

## Progress Tracking

TinkerBunker automatically tracks your progress as you navigate through the course.

| Indicator | Where It Appears | Description |
| -------------------- | ----------------------------- | --------------------------------------------------- |
| **Progress Bar** | Course card on Dashboard | Shows overall percentage of pages completed. |
| **Chapter Checkmark**| Sidebar within the course | Appears when all pages in a chapter are completed. |
| **Page Indicator** | Sidebar within the course | Marks individual pages as visited/completed. |

{% hint style="info" %}
Progress is recorded when you open a page. Make sure to scroll through or view the full content of a page for it to count as completed.
{% endhint %}

---

## Completing a Chapter

A chapter is considered **complete** when you have visited every page within it. Once all pages are marked as completed:

- A checkmark icon appears next to the chapter title in the sidebar.
- Your overall course progress percentage updates accordingly.

---

## Completing a Course

A course is considered **complete** when every chapter (and every page within each chapter) has been completed. Upon completion:

- The course card on your dashboard shows a **Completed** badge.
- You become eligible for a **Certificate** if the course has certificate generation enabled.
- Your stats are updated with the completion data.

{% hint style="info" %}
Even after completing a course, you can revisit any chapter or page at any time. Your completion status is preserved.
{% endhint %}

---

## Content Types on a Page

Pages can contain a variety of content types:

| Content Type | Description |
| ------------ | ---------------------------------------------- |
| **Text** | Rich text content with formatting. |
| **Images** | Inline images and diagrams. |
| **Videos** | Embedded video players. |
| **Embeds** | External content such as interactive widgets. |

---

## Tips for Effective Learning

{% hint style="info" %}
- Follow the chapters in order for the best learning experience, unless you are using the course as a reference.
- Check the **Stats** section periodically to monitor your proficiency and concept clarity.
- If a course has associated tests, take them after completing the relevant chapters to reinforce your understanding.
{% endhint %}
