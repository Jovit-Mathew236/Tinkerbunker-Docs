# Enrolling in Courses

TinkerBunker offers a rich catalog of courses that you can browse, preview, and enroll in. Courses can be **free** or **paid**, and access may be **public** or **private** depending on how the course creator configured them.

---

## Browsing the Course Catalog

1. Click **Courses** in the navigation bar.
2. You will see the course catalog with search and filter options.
3. Use the search bar to find courses by title or keyword.
4. Browse through category filters to narrow down results.

![Course Catalog](../.gitbook/assets/course-catalog.png)

---

## Course Visibility Types

| Visibility | Description |
| ----------- | ------------------------------------------------------------------------------------- |
| **Public** | Visible to all students in the catalog. Anyone can view the course details. |
| **Private** | Not listed in the catalog. Accessible only via a direct link or classroom assignment. |
| **Free** | No payment required. Enroll instantly with a single click. |
| **Paid** | Requires payment via Razorpay before access is granted. |

{% hint style="info" %}
A course can be both **Public** and **Paid** — it is visible to everyone but requires payment to enroll. Similarly, a **Private** and **Free** course is only accessible via a link but costs nothing.
{% endhint %}

## Enrolling in a Free Course

{% tabs %}
{% tab title="Steps" %}
1. Open the course detail page by clicking on a course card.
2. Review the course description, chapter list, and other details.
3. Click the **Enroll** button.
4. You will be enrolled immediately and the course will appear under **My Courses** on your dashboard.
{% endtab %}

## Enrolling in a Paid Course

Paid courses require a one-time payment processed securely through **Razorpay**.

{% tabs %}
{% tab title="Steps" %}
1. Open the course detail page.
2. Note the course price displayed on the enrollment button.
3. Click the **Enroll — ₹XXX** button.
4. A Razorpay payment modal will open.
5. Choose your preferred payment method (UPI, Credit/Debit Card, Net Banking, Wallet).
6. Complete the payment.
7. Upon successful payment, you are enrolled automatically and the course appears under **My Courses**.
{% endtab %}

{% tab title="Payment Methods" %}
| Method | Supported |
| -------------------- | --------- |
| UPI (Google Pay, PhonePe, etc.) | Yes |
| Credit Card | Yes |
| Debit Card | Yes |
| Net Banking | Yes |
| Wallets | Yes |
{% endtab %}
{% endtabs %}

{% hint style="warning" %}
If your payment fails or is interrupted, do not retry immediately. Check your bank statement first. If the amount was debited but enrollment did not complete, the refund will be processed automatically within 5-7 business days. Contact support if the issue persists.
{% endhint %}

## Course Detail Page

Before enrolling, you can review key information on the course detail page:

| Field | Description |
| -------------------- | -------------------------------------------------------- |
| **Title** | Name of the course. |
| **Description** | Overview of what the course covers. |
| **Chapters** | Number of chapters and a preview of the chapter list. |
| **Duration** | Estimated time to complete. |
| **Price** | Free or the amount in INR. |
| **Enrolled Students** | Number of students already enrolled. |

---

## Private Course Access

If a course is **private**, it will not appear in the public catalog. You can access private courses in two ways:

1. **Direct Link** — The teacher or administrator shares a URL with you.
2. **Classroom Assignment** — The course is linked to your classroom by your teacher, and it appears in your classroom view.

{% hint style="info" %}
If you receive a link to a private paid course, you will still need to complete the payment before gaining access to the content.
{% endhint %}

---

## After Enrollment

Once enrolled, the course appears in:

- **Dashboard → My Courses** section
- **Courses** page under the **Enrolled** tab

You can begin the course immediately by clicking on it.
