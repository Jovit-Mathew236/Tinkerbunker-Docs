# Certificates

TinkerBunker awards certificates upon course completion. Certificates are generated automatically, can be downloaded as PDFs, and are publicly verifiable using a unique link.

---

## How Certificates Are Earned

A certificate is generated when you **complete all chapters and pages** in a course that has certificate generation enabled.

| Requirement | Details |
| ---------------------- | ----------------------------------------------------------------- |
| **Course Completion** | All chapters and all pages within those chapters must be completed. |
| **Certificate Enabled**| The course must have the certificate feature enabled by the creator. |

{% hint style="info" %}
Not all courses issue certificates. If a course supports certification, it is typically mentioned on the course detail page before enrollment.
{% endhint %}

---

## Automatic Generation

Once you meet the completion criteria:

1. The system automatically generates your certificate.
2. A notification appears on your **Dashboard** confirming the certificate is ready.
3. The certificate includes:
   - Your name
   - Course title
   - Date of completion
   - A unique verification UUID
   - TinkerBunker branding

No manual action is required to trigger certificate generation.

## Certificate Contents

| Field | Description |
| ----------------------- | ---------------------------------------------------------------- |
| **Student Name** | Your registered display name on TinkerBunker. |
| **Course Title** | The full title of the completed course. |
| **Completion Date** | The date you completed the final page of the course. |
| **Verification UUID** | A unique identifier for this certificate, used for verification. |
| **TinkerBunker Logo** | Platform branding and certification mark. |

---

## Public Verification

Every certificate has a **unique UUID** that can be used to verify its authenticity.

### How Verification Works

1. Each certificate contains a verification link in the format:
   ```
   https://tinkerbunker.com/verify/{UUID}
   ```
2. Anyone with this link can visit it to confirm that the certificate is legitimate.
3. The verification page displays:
   - Student name
   - Course title
   - Completion date
   - Confirmation that the certificate is valid

{% hint style="info" %}
Share your verification link on your resume, LinkedIn, or portfolio to let employers and institutions verify your accomplishments instantly.
{% endhint %}

## Frequently Asked Questions

{% tabs %}
{% tab title="Can I get a certificate for every course?" %}
Only courses that have certificate generation enabled by the course creator will issue certificates. Check the course detail page for a certificate badge or mention.
{% endtab %}

{% tab title="What if my name is wrong on the certificate?" %}
The certificate uses your registered display name. Update your profile name before completing the course to ensure the correct name appears. If the certificate has already been issued, contact support for assistance.
{% endtab %}

{% tab title="Can certificates expire?" %}
Certificates issued by TinkerBunker do not expire. They remain valid and verifiable indefinitely.
{% endtab %}
{% endtabs %}

{% hint style="warning" %}
Ensure your profile name is accurate before completing a course. Certificate regeneration with a corrected name may require support intervention.
{% endhint %}
